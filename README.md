# Core-Banking-System

This is a **semester project** for the **Database Systems** course.  
It is a full-stack Core Banking System that manages customer accounts, transactions, and audit logs.  
Built with **Node.js** for the backend, **HTML/CSS/JavaScript** for the frontend, and **MySQL** for the database.

---

## Project Features

- Customer account management (create, update, view)  
- Transaction management (deposit, withdrawal, transfer)  
- Audit logs to track all activities  
- Frontend interface with responsive HTML/CSS pages  
- MySQL database with provided SQL scripts
